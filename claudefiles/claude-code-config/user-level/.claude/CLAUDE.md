# Personal Preferences (user-level)

## Language
- コードのコメント・コミットメッセージ・PR説明は日本語
- 変数名・関数名・型名は英語
- Claude との対話は日本語

## Model Strategy
- メインセッション: Opus 4.8（計画・設計・レビュー・意思決定）
- サブエージェント: Sonnet 4.6（実装・テスト・探索）
- Opus は自分でコードを書かず、必ずサブエージェントに委譲する

## Coding Style
- 正確さ最優先。不確実な情報は明示する
- 結論から話す
- 最新情報はウェブ検索で裏付けを取る
